# Size düşen nedir?

MongoDb başlangıç dökümanıdır.

Kodları çalıştırın,

    $ git clone git@github.com:seyyah/rails-mongodb.git
    $ cd rails-mongodb/
    $ bundle
    $ rails s --binding=1.2.3.4 --port=3000

Test: http://1.2.3.4:3000/expenses
